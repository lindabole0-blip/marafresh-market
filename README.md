# maraFresh Market
maraFresh Market is a simple e-commerce website designed to allow users to browse fresh farm products and place orders online. Adjustments is being made to allow farmers to create their profiles and sell whatever they are planting.
## Current Features
- View different fresh farm products
- Add products to cart
- Order page showing product prices
- Search functionality
- Navigation bar for easy movement across the page
- Login and account creation paged
